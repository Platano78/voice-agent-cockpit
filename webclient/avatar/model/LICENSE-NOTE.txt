Per the same TalkingHead README section:

  Example avatar "avatarsdk.glb" was created at AvatarSDK (https://avatarsdk.com) for
  non-commercial use.

Verified before adoption: loads via TalkingHead's showAvatar() with no console errors,
and its morph target dictionary includes ARKit-style viseme/blendshape keys (browDownLeft,
eyeBlinkLeft, etc.) required for lip-sync. Uses its own mesh/material naming convention
(AvatarBody/AvatarHead/AvatarLeftEyeball/AvatarRightEyeball/outfit_top, not RPM's Wolf3D_*)
— see avatar/avatar.mjs SLOT_TO_MATERIAL for how both naming conventions are mapped to the
same semantic material slots (hair/skin/top/eyes) used by themes.json.

Same license tier as brunette-t.glb (non-commercial) — placeholder/personal-use asset.
